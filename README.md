# QLC — Que Le Compramos | Website

Static HTML/CSS site for [quelecompramos.com](https://quelecompramos.com). No build step, no dependencies.
Auto-deploys to Vercel on every push to `main`.

## Files

```
qlc-site/
├── index.html        — Main landing page
├── privacidad.html   — Privacy policy
├── contacto.html     — Contact form
└── styles.css        — Shared stylesheet (mobile-first)
```
