Tu privacidad

La sorpresa está protegida.
Tus datos, también.

Los datos personales que compartís (email, teléfono, fecha de nacimiento) se guardan cifrados y nunca aparecen en ninguna respuesta del agente. Se usan únicamente para identificar personas dentro de tu grupo.

✓
Nadie puede ver su propia lista

La persona agasajada no puede consultar los regalos que el grupo está planeando para ella.

✓
Datos cifrados desde el primer día

Emails, teléfonos y fechas de nacimiento se almacenan con cifrado AES-256. Nunca se muestran.

✓
Eliminación inmediata a pedido

Cualquier usuario puede pedir que se borren los datos de una persona con un solo mensaje, en cualquier momento.

✓
Sin publicidad. Sin venta de datos.

Tus datos no se comparten con nadie ni se usan para fines comerciales.
